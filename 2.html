Détails du projet

Ce projet consistait à créer une base de données dédiée à la qualité de l'air. J'ai développé un Modèle Conceptuel de Données (MCD) pour définir les entités et leurs relations. Ensuite, je suis passé à un Modèle Logique de Données (MLD), où j'ai transformé le MCD en tables adaptées à un système de gestion de bases de données relationnelles. Finalement, j'ai créé un Modèle Physique de Données (MPD) qui représente la structure réelle de la base de données.

J'ai réalisé la base de données sur JL Merise, un outil de modélisation très efficace. Cela m'a permis de visualiser et de schématiser le projet de manière précise. Ensuite, j'ai pu interroger cette base avec des requêtes SQL pour extraire des informations pertinentes. Ce projet m'a apporté une expérience précieuse en modélisation de données et en gestion des bases de données.
