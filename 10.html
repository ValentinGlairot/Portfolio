Site Web Full Stack Sécurisé

Détails du projet

Ce projet consiste en la création d’un site web complet (frontend + backend) avec des fonctionnalités de sécurité avancées. Nous avons développé pour apprendre à sécuriser une architecture web tout en respectant les bonnes pratiques du développement moderne.

  • Frontend : HTML, CSS, JavaScript
  • Backend : PHP / PHP Unit
  • Base de données : SQL / PhpMyAdmin
  • Authentification sécurisée avec Hash Mot-de-passe
  • Création Vhosts pour accès local
  • Validation des entrées et gestion des erreurs
  • Protection contre les attaques XSS / injections SQL avec requête préparée
