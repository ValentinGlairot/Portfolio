Station Météo avec Arduino

Détails du projet

Ce projet consiste à concevoir une station météo utilisant Arduino. Les données de capteurs pour l'humidité, la température, la luminosité et l'heure (via un module RTC) sont collectées et enregistrées sur une carte SD.

Les points clés de ce projet incluent :

  • Utilisation d'une base shield pour connecter les capteurs.
  • Stockage des données sur une carte SD pour une analyse ultérieure.
  • Intégration d'un module RTC pour l'horodatage précis des mesures.
