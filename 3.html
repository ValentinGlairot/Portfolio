Modulation ASK & FSK en Python

Détails du projet

Ce projet vise à implémenter la modulation et la démodulation de signaux à l'aide des techniques ASK et FSK. Ces deux techniques permettent d'encoder des informations binaires dans des signaux analogiques.

Dans ce projet, la modulation permet de transmettre des données sous forme de signaux, tandis que la démodulation permet de récupérer les données reçues. Une des principales caractéristiques de ce projet est l'implémentation de l'encodage Manchester, qui permet de garantir une synchronisation précise des données transmises.

Les fonctionnalités clés du projet incluent :

  • Modulation ASK pour la transmission de données via l'amplitude du signal.
  • Modulation FSK pour la transmission de données via la fréquence du signal.
  • Encodage Manchester pour assurer l'intégrité des données et la synchronisation.
  • Démodulation des signaux pour récupérer les informations initiales après transmission.
